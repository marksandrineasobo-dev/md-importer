# Markdown Importer

A lightweight WordPress plugin scaffold for importing Markdown files into draft posts.

## Features

- Admin import screen under Tools > Markdown Importer
- Upload `.md` or `.markdown` files
- Converts headings, bold, italic, and blockquotes into HTML
- Creates a draft post from the uploaded Markdown file

## Installation

1. Place the `md-importer` folder in `wp-content/plugins/`
2. Activate the plugin in the WordPress admin
3. Open `Tools > Markdown Importer`
4. Upload a Markdown file to create a draft post
